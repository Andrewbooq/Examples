# .NET Standard 2.1

## Задание

На .NET Standard 2.1 создать свой словарь аналогичный System.Collections.Generic.Dictionary, с теми же методами, но со следующими требованиями: 

В качестве ключа TKey словарь принимает исключительно классы, унаследованные от IdentifiedObject (приложено), класс TValue – любой.
Метод получения значения по ключу должен работать не менее, чем на 30% быстрее, чем в System.Collections.Generic.Dictionary при общем количестве значений в словаре ~20

## Результат

```
Универсальный класс Dictionary<TKey,TValue> предоставляет сопоставление из набора ключей с набором значений. Каждое дополнение к словарю состоит из значения и связанного с ним ключа.
Получение значения с помощью ключа очень быстро, близко к O(1), так как класс Dictionary<TKey,TValue> реализуется как хэш-таблица.
Примечание
Скорость извлечения зависит от качества хэширования алгоритма типа, указанного для TKey.
```
(c) https://learn.microsoft.com/ru-ru/dotnet/api/system.collections.generic.dictionary-2?view=net-8.0
